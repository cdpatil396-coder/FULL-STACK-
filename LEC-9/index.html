ID & Classes

ID An ID is an attribute, a unique identifier assigned to only one HTML element within a page. it is often used for unique styling and javascript manipulations
Classes The class attribute lets you give the same name to multiple HTML elements. that way, you can easily change their look or behaviour all at once . Classes are not unique and can be assigned to multiple elements. they are generally used for applying the same styles or behaviours to a group of elments
